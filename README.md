
# 水韵江苏配套数字资源

## 项目简介
本项目是河海大学《水韵江苏》课程的配套数字资源平台，旨在通过数字化手段提升江苏文化学习体验。平台包含知识浏览、问答互动、旅行地图等功能模块。

## 功能模块
- **知识E览**：智慧教材与AR增强现实内容
- **知苏答理**：知识问答与学习挑战
- **苏行漫迹**：江苏文化旅行地图与沉浸式学习

## 技术栈
- HTML5/CSS3
- JavaScript (ES6+)
- IonIcons 5.5.3 图标库
- Vite 4.4 构建工具

## 快速开始
```bash
# 克隆仓库
git clone https://github.com/your-repo/syjs_mobile.git

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```
